<h1 align="center">Χαλαρό</h1>

<p align="center">一个仅供学习自用的PC端动漫视频网站</p>


---

## 技术栈

- 🍭 vue3+pinia
- 🌍 vite
- 💪 typescript
- 💪 less
- 📖 video.js
- 📖 element-plus

## 功能

- 首页推荐（热门、更新、新番动态、其他推荐）
- 搜索
- 多源播放
- 记录（历史、进度）
- 自定义主题
- 可变动数据源

## 浏览器支持

建议使用chrome内核的浏览器，如`Chrome`、`Edge`、`FireFox`（暂未做兼容性处理）
